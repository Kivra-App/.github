What is Kivra?

Kivra preserves the development knowledge that usually disappears after a project is finished.

Instead of remembering only what changed, Kivra helps developers remember why it changed.

Preserve

* 🌳 Project Structure
* 📦 Project Metadata
* ▶️ Command History
* 📜 Terminal Logs
* 🐞 Error History
* 📝 Resolution Notes
* 🧠 Project Memory
